# Portfolio Website

This is my personal portfolio website built using HTML, CSS, and JavaScript.  
The main purpose of this project is to practice frontend development and showcase my skills, projects, and contact details.

---

## Features

- Responsive design for desktop and mobile
- Modern dark-themed UI
- Smooth navigation with JavaScript
- Skills and projects showcase
- Interactive contact form (frontend only)
- Mobile hamburger menu

---

## Technologies Used

- HTML5 – Structure of the website  
- CSS3 – Styling, layout, animations, responsiveness  
- JavaScript – Interactivity and user experience  

---


---

## Responsive Design

The website is fully responsive and works well on:
- Desktop
- Tablet
- Mobile devices

Flexbox and CSS Grid are used to build responsive layouts.

---

## Contact Form

The contact form currently works on the frontend only:
- JavaScript is used to validate empty fields
- A success message is shown after submission
- No backend integration is added yet

---

## Purpose of This Project

- Practice HTML, CSS, and JavaScript
- Improve frontend development skills
- Learn responsive design and UI/UX basics
- Prepare a foundation for future projects
